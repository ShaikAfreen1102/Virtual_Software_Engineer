# Virtual Software Engineer (VSE)

Virtual Software Engineer (VSE) is a role-based learning and simulation platform
designed for final-year engineering students and freshers to experience
real-world software development workflows without internships.

The platform simulates how engineers work in real companies by guiding users
through the complete Software Development Life Cycle (SDLC) using tasks,
visual learning, time-bound phases, quizzes, and certification.

## ğŸ“š Learning Rules & Validation

To maintain learning integrity, the platform enforces strict rules:

- Users must spend the required minimum time on each learning phase
- Tasks unlock sequentially (no skipping allowed)
- Videos are supported with image-based visual fallbacks
- A mandatory quiz of 10 MCQs must be completed after task completion
- Only 3 incorrect answers are allowed
- Certificate download is enabled only after passing the quiz

This ensures users genuinely learn instead of rushing to certification.
